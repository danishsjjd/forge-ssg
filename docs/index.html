Forge

I built a Markdown → HTML static site generator from scratch. This website is powered entirely by forge-ssg.

If you're reading this page, the site generator is already working.

About This Project

forge-ssg is a simple static site generator that converts Markdown files into HTML pages.

Example Code

# Hello World

This page was written in **Markdown**.

- Simple
- Fast
- Easy to read

The generator turns this into HTML during the build process.
